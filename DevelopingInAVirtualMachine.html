Developing In A Virtual Machine

Introduction

In this lab I explored how a virtual machine can be used as a development environment for creating and editing websites. The aim of the lab was to understand how developers can work remotely inside a virtual environment while testing and updating projects.

The lab focused on development workflows, editing files remotely and understanding the difference between development and deployment servers.

Development Environment

A second virtual machine was used as a development server environment. This virtual machine allowed website files to be edited and tested before deployment.

Using a separate development environment helps reduce the risk of breaking the live website because changes can be tested safely before deployment.

This demonstrated how real-world software projects often separate development systems from production systems.

Using Visual Studio Code

Visual Studio Code was installed inside the virtual machine and used to edit the portfolio website files.

HTML, CSS and JavaScript files were modified directly inside the virtual machine environment. This made it possible to test updates remotely and simulate a professional development workflow.

Through this process I gained more experience using development tools inside remote systems.

Editing And Testing The Website

Changes were made to the portfolio website inside the development virtual machine and then tested through the browser.

This process demonstrated how updates can be made safely in a testing environment before moving changes to a deployment server.

I also learned about file permissions and administrator privileges when editing protected server directories.

Development Vs Deployment Servers

The development virtual machine was used for editing and testing changes, while the deployment virtual machine was used for hosting the live version of the website.

Separating development and deployment systems is important because it allows developers to test updates without affecting the live website.

This concept is widely used in professional software development and web hosting environments.

Reflection

This lab improved my understanding of remote development environments and virtual machine workflows. I learned how development and deployment systems can work together to support safer and more organised software development.

I also improved my understanding of remote file editing, development tools and server management concepts.
