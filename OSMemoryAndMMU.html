OS Memory And MMU

Introduction

In this lab I explored memory management concepts used by operating systems and how the Memory Management Unit (MMU) translates addresses between virtual memory and physical memory.

Modern operating systems use memory management techniques to improve efficiency, multitasking and process isolation. During this lab I examined paging, address translation, page tables and virtual memory systems.

Paging

Paging is a memory management technique that divides memory into fixed-size pages and frames. Virtual memory is divided into pages while physical memory is divided into frames of equal size.

During the exercises I calculated page numbers, offsets and frame mappings using different memory sizes and address values.

Paging improves memory management because processes do not need to occupy continuous blocks of memory.

Address Translation

The MMU translates virtual addresses into physical addresses using page tables. This process allows programs to use virtual memory without directly accessing physical memory locations.

In the exercises I worked with binary and hexadecimal address conversions and calculated how addresses are translated between memory systems.

Address translation is important because it allows operating systems to safely manage memory between multiple processes.

Virtual Memory

Virtual memory allows operating systems to use secondary storage as additional memory space when physical RAM becomes full.

During the lab I explored how virtual memory improves multitasking and allows large programs to run efficiently.

I also learned how page faults occur when a required page is not currently loaded into physical memory.

Memory Management Unit (MMU)

The Memory Management Unit is responsible for converting virtual addresses into physical addresses during program execution.

The MMU works closely with page tables and the operating system to ensure that memory is allocated correctly and securely.

Understanding the MMU is important because it forms the foundation of modern operating system memory management.

Reflection

This lab improved my understanding of memory management techniques and virtual memory systems used in operating systems.

I also improved my understanding of paging, address translation and how the MMU controls memory access between processes.
